Program metadata
==========================================

This is a Scala project with SBT.
java version 1.8 or above is required for compilation and Execution, Along with scala version 2.13.1


==========================================

Steps to run the program

1>Execute the file from sbt shell , 
use the command 
    clean 
    compile
    run 
the data input for the application could be found at "triangle.txt" at the root of the folder
Output could be founds at the root folder of the project with the name Output.txt , which clearely shows the result

2> The test cases has been written inside src/test/scala/MaxCountTest.Scala
   The Input has been hard-coded as a 2d-Array with Int values.
   To Execute the test cases from sbt shell using the command
   test , after clean and compile.
